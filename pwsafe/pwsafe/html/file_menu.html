File Menu

The File Menu contains options related to managing and creating new password safe databases.

New Database

Create a new blank password safe database.

Open Database

Open an existing password safe database.

Recently Used Files

This section of the file menu may list the recently opened database files. This section will be blank if this is the first time Password Safe has been used.

Notes:

  • The list of recently opened files may be cleared by selecting "Clear Recent database List"
  • The number of recent files that Password Safe will keep may be configured via the Manage->Options->Display dialog box

Save

Save any changes to the current safe. If no changes have been made since the previous save, this choice will be greyed-out.

Save As

Save a copy of the current safe under a new name or location.

Export To

Export the contents of the current database to an external file in either an old Password Safe database format or as text. See here for more information on exporting the database to other formats.

Import From

Import the contents of a previously exported or correctly formatted text file or a KeePass password database into the current database. See here for more information on importing text files.

Merge Database

Merge a Password Safe database or backup file (.bak) into the current database.

Compare Database

Compare the currently opened database with another.

Note: Both the Merge and Compare operations may take a long time, especially for large databases.

Exit

Closes the Password Safe program.
