Haciendo y restaurando copias de seguridad

Puesto que la información almacenada en la base de datos de contraseñas es usualmente sensible e importante, Password Safe proporciona varios mecanismos para hacer y restaurar copias de seguridad de la bases de datos. Esta sección describe como hacer copias de seguridad de tu base de datos de contraseñas. Password Safe puede configurarse para hacer copias de seguridad automáticas de tu base de datos cada vez que se guarda. Esas copias de seguriad se llaman copias de seguridad intermedias, y están descritas aquí.

Hacer una copia de seguridad

Crear una copia de seguridad de la base de datos de contraseñas actual.

  1. Pulsar en Hacer copia de seguridad
  2. Selecciona una ubicación para el archivo de respaldo
  3. Pulsa Guardar

Nota: Es muy recomendable que hagas copias frecuentes de tus base de datos de contraseñas. Esto ayudará a salvaguardar la lista de contraseñas ante caídas de sistemas o corrupción de ficheros. La copia de seguridad está encriptada usando la misma contraseñas que la base de datos actual de que se hace la copia de respaldo.

Restaurar desde una copia de seguridad

Restaurar un archivo de base de datos de contraseñas.

  1. Pulsa Restaurar desde copia de seguridad
  2. Ir a la ubicación de la copia de respaldo y seleccioan el archivo a restaurar
  3. Puls Abrir
    Se preguntará por la contraseña.
  4. Introducir la base de datos de las contraseñas y pulsar Abrir
Notas:
  • La base de datos restaurada no se "guarda" con la extensión estandar de base de datos de Password Safe (.psafe3) hastaa que no seleccionas Guardar o Guardar como desde el menú archivo. Se pedirá "Guardar" la base de datos y darle un nombre name si no la has grabado antes de salir de la aplicación.
  • Adicionalmente a la copia y restauración definida aquí, Password Safe guarda una copiad de la base de datos antes de grabar un cambio. La copia se guarda en el mismo directorio de la base de datos actual, con el mismo nombre, excepto la extensión, que es ".ibak" en lugar de ".psafe3".
