Crear y abrir una base de datos

Password Safe permite a los usuarios almacenar todas sus contraseñas en una única "caja de seguridad" (base de datos de contraseñas), o crear multiples bases de datos para diferentes propósitos (por ejemplo, una para el trabajo, otra para uso personal). Cada base de datos es independiente se puede mover y utilizar en diferentes sistemas, siempre que esté instalada la misma versión de Password Safe. Las bases de datos se encriptan con una clave derivada de la contraseña maestra(la contraseña maestra no se guarda en la base de datos de ninguna forma).

Si es la primera vez que usas Password Safe, pulsa el botón Nueva base de datos. Se te pedirá elegir el nombre y la ubicación de tu base de datos de contraseñas (el valor predeterminado, "pwsafe.psafe3", está bien). Posteriormente, se te pedirá que introduzcas una contraseña maestra que se utilizará para cifrar y bloquear el contenido de su nueva caja fuerte.

Crear una nueva base de datos

Hay dos formas de crear una nueva base de datos. Pulsando Nueva base de datos desde el diálogo Introducir contraseña maestra cuando se inicia el programa, o desde el menú Archivo > Nueva... cuando Password Safe ya está abierto. Utilizando cualquiera de los métodos hace aparecer el diálogo Configuración de contraseña maestra.

Nota: Por defecto, Password Safe mostrará la última base de datos que se utilizó, y pedirá la contraseña para abrirla. Si usas varias bases de datos, puedes elegir entre ellas usando la lista desplegable. Alternativamente, puedes escribir la ruta de acceso a la base de datos, o seleccionarla del cuadro de diálogo Archivo pulsando el botón de puntos suspensivos ("..."). Una vez que una base de datos está abierta, puede cambiar a otra a través de la opción Abrir... del menú Archivo.



Teclado virtual

Para configurar una contraseña maestra segura, puedes usar caracteres (por ejemplo en un teclado alemán) como ä, ö o similar. Para introducir esos caracteres, necesitas un teclado de otro idioma. Ese teclado está disponible como teclado virtual. Se utiliza para simular todos los caracteres disponibles. El teclado virtual se muestra pulsando el símbolo . Los caracteres que no están disponibles en el teclado físico, pueden introducirse de esta forma. Así puedes fijer la contraseña maestra que desees. Incluso partes de la contraseña maestra pueden introducirse de esta manera.
