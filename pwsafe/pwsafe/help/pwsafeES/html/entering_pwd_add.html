Nombres de usuario, contraseñas y grupos: Adicional

Nombres de usuario, contraseñas y grupos junto con los parámetros adicionales se gestionan en 4 pestañas:

  • Básico: todos los elementos necesarios para la gestión de contraseñas
  • Adicional: especificaciones adicionales, incluyendo el historial
  • Fechas y horas: caducidad y estadísticas
  • Política de contraseñas: diferencias específicas sobre los valores predeterminados

Pestaña adicional


  • Autoescritura: Si este campo no está vacío, su contenido reemplaza el valor por defecto para la función de autoescritura. Para más información, ver el tema de ayuda "Autoescritura".
  • Ejecutar Cmd: Este campo puede usarse par indicar un comando o programa a ejecutar en la acción 'Ejecutar comando' (en el menú editar, menú contextual o Ctrl-R). Password Safe puede pasar argumentos al comando, tales como el usuario, contraseña, etc, tal como se describe aquí.
  • Acción al hacer doble clic: Esto te permite definir que acción realizará PasswordSafe al hacer doble clic sobre una entrada. Por defecto, se copia la contraseña de la entrada al teclado. La acción por defecto se fija en Herramientas->Opciones->Misc..
  • Historias de contraseñas: Se puede configurar Password Safe para guardar las últimas contraseñas de una entrada. Este puede hacerse al crear una entrada o al editar una entrada existente. Puees ver las contraseñas recientes relacionadas con la entrada. Puedes cambiar el número de contraseñas recientes a guardar. Para más información ver la página de historial de contraseñas.
