Créditos del programa

Agradecimientos a Bruce Schneier por crear el algoritmo Twofish y por liberar Password Safe al público; y a SourceForge.net por haber acogido el proyecto y el apoyo a la comunidad Open Source.
La versión 3.06 y posteriores de Password Safe utiliza TinyXML para analizar el archivo de configuración de la aplicación. El código de TinyXML se proporciona bajo la licencia zlib.
La versión 3.12 y posteriores de Password Safe utilizan un algoritmo creado por Tom Van Vleck, Morrie Gasser, and Dan Edwards para generar contraseñas pronunciables.
