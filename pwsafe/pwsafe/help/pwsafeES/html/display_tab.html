Pestaña visualizar

Las funciones de la Pestaña visualizar controla como se muestra Password Safe y las entradas de contraseñas.

Mantener Password Safe siempre visible

Estando seleccionado, Password Safe estará siempre sobre el resto de ventanas mostradas si no está minimizado, sin importar que ventana tiene el foco.

Mostrar usuario en vista de árbol

Estando seleccionado, los nombre de usuario se mostrarán entre corchetes siguiendo al nombre del elemento en la vista árbol jerarquizado.

Mostrar contraseñas en vista de árbol

Estando seleccionado, las contraseñas se mostrarán entre llaves siguiendo al nombre de usuario en la vista árbol jerarquizado.
Esta opción está altamente desaconsejada en entornos donde otros pueden ver el monitor.

Mostrar notas como menú desplegable en vistas en árbol y lista

Estando seleccionado, se mostrarán las notas al señalar un elemento.

Mostrar contraseña en agregar y editar

Estando seleccionado, las contraseñas en el dialogo añadir y editar/ver entrada serán visibles por defecto. Puedes mostrar u ocultar la contraseña de una entrada pulsando el botón "Mostrar/Ocultar".

Mostrar notas al editar

Estando seleccionado, las notas se mostrarán por defecto en el dialogo editar. Si no lo está, debes hacer clic en el campo notas para ver el texto que contiene.

Ajuste de línea en notas al añadir y editar

Estando seleccionado, las palabras se ajustarán automáticamente en la ventana, sin barra de desplazamiento horizontal.
Si no está seleccionado, aparecerá una barra de desplazamiento horizontal si es necesario.

Mostrar líneas de cuadrícula en la vista de lista

Estando seleccionado, se mostrará una línea de cuadrícula en la lista, haciendo que parezca una hoja de cálculo.

Poner grupos primero en vista de árbol

Por defecto, las entradas se muestran en orden alfabético en la vista de árbol. Cuando se selecciona esta opción, todos los grupos se mostrarán primero (en orden alfabético), seguido por el resto de entradas del nivel superior.

Resaltar entradas modificadas

Mientras las entradas modificadas no se han guardado, aparecerán resaltadas. Si has seleccionado "Guardar base de datos inmediatamente tras editar o añadir" en la pestaña copias de respaldo, las entradas nunca se resaltarán ya que se guardan inmediatamente.

Avisar N días antes que las contraseñas caduquen

Estando seleccionado, PasswordSafe revisará todas las entradas al cargar la base de datos y mostrará una lista de aquellas cuya fecha de caducidad es N dias o menos desde la fecha actual.

Vista inicial en árbol

Determina que las entradas se mostrarán en vista de árbol al abrir la base de datos.

Icono inicial en área de notificación

Esto le permite cambiar el color del icono de la bandeja del sistema que aparece cuando Password Safe arranque por primera vez y cuando no hay base de datos abierta.
