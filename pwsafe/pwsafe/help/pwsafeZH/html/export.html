输出项目

Password Safe 支持已存储密码输出(和输入)到(从)若干格式文件。 需要删除数据使用文件 -> 输出菜单。支持格式为:

  • Password Safe v1 (适用版本为 1.x 的 Password Safe)
  • Password Safe v2 (适用版本为 2.x 的 Password Safe)
  • 纯文本文件
  • XML

输出适用于早期版本的 Password Safe 文件，只需选择想要的格式和指定希望创建的数据库文件名称。

输出文本

备注: 输出文本为所指定密码项目的一个无保护副本。在不充分使用后请删除该文件副本，使攻击程序难以获得及有效的绕过安全程序。
请最好不要使用该操作，除非你已了解和接受该风险！

Password Safe 允许输出密码项目为文本文件。该文件可能因此被使用或被其它程序修改。
将首先询问输入主密码或让确认可能输出数据库为一个未加密的文本文件。

同样可以指定部份选项:

  • 密码项目将被输出字段之间存在标记的文件。例如: 在组、标题、用户名、密码和备注字段之间。“组/标题”考虑为单字段。 任何字段包含一个空白时将被输出在双引号字符里。 只有备注字段可以包含多行项目。正常处理每个备注行位置在输出文本字段里使用一个分隔行。 不过，这样使得它难以分类或使用比较两个不同输出文件时。要克服这个可以在输出它们时指定一个多行定界符分隔。 如果转到输入一个文本文件创建具有这样一个定界符，那么它必须被指定， 所有备注行将作为一个单独行出现在数据库结果里。
    如果必要，标记想要标记的所有分隔行于备注字段的一行上然后输入符号用作定界符。 由于备注字段整体被双引号标注，该符号禁止被选为多行定界符。

  • 更多控制输出格式可以通过单击“高级”按钮。将出现下列对话框:

该对话框允许只输出基于文本指定字段项目的部份和/或指定哪些字段被输出。 在选定完成时单击确定或取消不作出任何更改返回到输出文本对话框。

此时将询问用户提供新文件名称以及如果文件已存在所指定的目录时确认覆盖同名文件。

备注: 详细的输出文本文件格式资料被建立在“纯文本文件输出与输入格式”节段。

XML

XML (可扩展置标语言)是一种流行格式作为应用程序之间传递数据。Password Safe 可以输出(和输入)数据库项目为这种格式。 构造(或规划) 以输出数据和验证定义为标准的文件pwsafe.xsd， 是提供作为布局的一部份。该文件被用于 Password Safe 在输入输出 XML 数据时。
备注: 输出文件为选定密码项目的一个无保护副本,请记得在不使用后及时删除它， 以防黑客入侵并有效绕过安全程序轻松获取其中敏感信息。该文件可以使用其它文本编辑器轻易打开，比如: Windows 记事本！
请最好不要使用该操作，除非你已了解和接受该风险！
显示的 XML 文件是使用 pwsafe.xsl(可扩展样式语言文件)表示的。 它是一个样品文件，可以根据你自己需要作出修改，提供作为一部份布局。 该文件只是一个样品和你可以轻易修改它显示你想要显示的任何字段。 虽然只作为显示数据和即使不修改它，但在稍后你将依旧可以输入它到 Password Safe 中。

该对话框允许你只输出部份项目， 基于指定字段的文本和/或指定哪些字段被输出。完成时单击确定或取消返回到输出 XML 对话框不作出任何更改。 请注意 XML 规划需要标题和密码字段，因此这些不能被取消选定。
