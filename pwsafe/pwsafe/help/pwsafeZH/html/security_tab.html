安全选项卡

安全选项卡提供用户访问控制安全设置。其易用性和程序安全受这里列出的选项影响，但根本上不参与加密密码数据库。

最小化时清空剪贴板

勾选此项，当应用程序最小化时程序将清空剪贴板中的所有存储数据内容 (如密码)。 这是预防其他任何人或任何其它程序从剪贴板中读取内容。注意选择该选项对于在杂项选项卡中双击操作“复制密码到剪贴板并最小化”选项将导致意外结果(双击项目时密码虽然被复制，但最小化时却又被立即清空掉)。

退出程序时清空剪贴板

勾选此项，当界面被最小化或关闭时程序将清空剪贴板中的所有存储数据内容。这是预防其他任何人或任何其它程序从剪贴板中读取内容。

确认项目复制到剪贴板

勾选此项时，显示一个提示对话框要求用户每次从数据库复制密码到剪贴板时进行确认。

最小化时锁定密码数据库

勾选此项，当程序最小化时自动锁定密码数据库，而当再次最大化访问程序时必需输入密码。

锁定密码数据库在工作站界面

勾选此项将导致密码数据库空闲在规定的时间后被锁定在工作站界面。目的是预防未经用户授权访问时访问工作站。

密码数据库空闲超时后自动锁定...

勾选此选项和设置一个时间将导致 Password Safe 在所设定间隔时间过后还没有用户操作应用程序时自动锁定密码数据库。
