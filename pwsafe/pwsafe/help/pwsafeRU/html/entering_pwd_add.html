Имена пользователей, пароли и группы: дополнительно

Имена пользователей, пароли и группы, а также другие параметры настраиваются на четырёх вкладках:

  • Основные: все необходимые элементы, связанные с управлением паролями
  • Дополнительно: дополнительные характеристики, включая историю паролей
  • Дата и время: дата истечения пароля и статистика
  • Политика создания паролей: переопределяет для данного элемента «политику по умолчанию»

Вкладка «Дополнительно»


  • Автонабор: если поле не пусто, его содержимое заменяет на параметры автонабора, установленные по умолчанию (значение по умолчанию указано сверху и берётся из настроек Password Safe для текущего контейнера, задаваемых на вкладке Управление→Настройки→Разное). Для получения дополнительной информации см. Автонабор.
  • Выполнить: данное поле используется для указания команды/программы, выполняемой при выборе пункта «Выполнить команду» (в меню «Изменить» или контекстном меню) или по горячей клавише Ctrl+R. Password Safe может передать в качестве аргументов имя пользователя, пароль и содержимое других полей, описанных здесь.
  • Действие по двойному клику: задаёт действие, которое Password Safe выполнит при двойном щелчке на элементе. Действием по умолчанию является копирование пароля в буфер обмена. Действие по умолчанию задаётся на вкладке Управление→Настройки→Разное.
  • История паролей: Password Safe может сохранять несколько предыдущих значений пароля. Настройку можно задать при создании или редактировании элемента. В списке отображаются предыдущие пароли для указанного элемента. Количество сохраняемых предыдущих паролей может быть изменено позже. Для получения дополнительной информации см. История паролей.
