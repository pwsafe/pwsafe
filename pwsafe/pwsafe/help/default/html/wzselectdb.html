Wizard Select Database or File

Compare, Merge or Synchronize

When performing the database Compare, Merge or Synchronize function, you must provide the name of the database that you wish to Compare with the currently open database, Merge into the currently open database or Synchronize with the currently open database.

The password requested is that of this new database and not the currently open database.

Export to Text or XML

When performing an Export of the current database to a clear Text or XML file, you must provide the name of the file to create or overwrite (you will be asked to confirm if you wish to replace an existing file).

The password requested is that of the currently open database.

You should also provide the character to be used to denote a line break in the Notes field so that it can be correctly processed when imported. When exporting to a plain text file, this character is also used to replace a dot (.) in a Title field so that the correct Tree structure is rebuilt on Import (note a dot is used to separate groups).

Advanced field selection criteria

In all cases, you have the option to select the "Advanced field selection criteria", which will show an extra Advanced page allowing you to restrict what these functions do based on the contents of the Group, Title, User Name, URL or Notes fields and/or which fields to Compare, Merge, Synchronize or Export.

Note 1: When exporting a single entry via the Edit Menu or Context Menu (by right clicking on the entry), the option to restrict the export based on the contents of the Group, Title, User Name, URL or Notes fields is obviously not available.

Note 2: When Merging a database into the currently open database, you will not be able to restrict which fields of the entries being merged are copied. If an entry passes your selection criteria, then the complete entry will be merged into the current database.
