Appendix: Keyboard Shortcut/Accelerator keys

These key combinations have been tested on a 102-key extended UK keyboard. The standard characters should work on any other keyboard with Latin characters. Some key combinations may be different on these keyboards - for example on a UK keyboard the the # symbol has its own key (with the tilde ~) but on a US keyboard it is above the 3 key (i.e. Shift+3). Please read this table and translate to your specific keyboard as appropriate - you can always experiment to see if a particular key combination works for you.

Depending on your translation, certain keys with the Alt key are reserved. These correspond to the letters used to invoke the main menu. Pressing the Alt key in the main display will show the these characters as underlined. For English, these would be Alt+F (File), Alt+E (Edit), Alt+V (View), Alt+M (Manage) and Alt+H (Help). For non-English translations, please refer to those letters underlined on the main menu to determine what will be reserved for you. For example, in the German translation "File" menu becomes "Datei" and so Alt+D would be reserved instead of Alt+F etc.

The Escape key is also reserved as it can be used to close the Find Toolbar or to minimise Password Safe to the System Tray. It is also the equivalent to the Cancel button on any dialog.

Some keys are reserved by Windows - for example Ctrl+Alt+Delete; Tab; Shift+Tab; Alt+Tab; Alt+Shift+Tab etc.

Non-standard keys are not supported e.g. those only on a Kanji keyboard.

Password Safe shortcut setting does not recognise the difference between the Alt, Ctrl and Shift keys on the left side of the keyboard from those on the right. This may make some key combinations unavailable on some keyboards. For example, the Euro symbol on a UK 102 extended keyboard is the right Alt key (Alt Gr)and the 4 key. However, pressing this combination results in a shortcut of Ctrl+Alt+4. This is the way Microsoft's HotKey processing works and is not a Password Safe restriction.

Note: Setting non-standard keys may make these particular shortcuts unavailable if you then use another computer with a keyboard that does not have these keys. It is recommended that you only select combinations from those listed here.
