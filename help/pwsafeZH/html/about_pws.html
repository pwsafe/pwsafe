关于 Password Safe

Password Safe 使用Bruce Schneier 和 Counterpane 实验室原先制定和公布的 Twofish 加密算法。

Password Safe 目前是一个开源项目，托管在 sourceforge.net。最新的程序更新、文件和新闻位于 http://pwsafe.org。

Password Safe 可以免费获得，并根据开放源码倡议(OSI)中规定分配的限定标准制作了“典雅许可证 2.0”，该许可证副本包含在 Password Safe 安装程序包中，文件名称为英文 LICENSE。

Twofish 是一种快速，免费替代 AES、DES 和 IDEA 的加密算法。Twofish 算法的详细资料，包括速度比较和广泛的产品清单，使用 Twofish 等信息，可见于 http://www.schneier.com/twofish.html.

请从安装目录文件夹中阅读 readme.txt、changelog.txt 和 releasenotes.txt 文件的有关程序最新更改、bug 修复和文档附录等信息。
