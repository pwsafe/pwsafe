文件菜单

文件菜单包含涉及管理与创建新 Password Safe 数据库选项。

新建

新建一个空白 password safe 数据库。

打开

打开一个现有 password safe 数据库。

关闭

在不退出应用程序的情况下只关闭当前已打开的 password safe 数据库。

最近保险箱

该文件菜单部份可列出最近打开过的数据库文件。如果是首次使用 Password Safe 该部份将为空白。

备注:

  • 最近打开文件列表可通过选择“清除最近保险箱列表”清除。
  • 最近 Password Safe 文件保存数量可通过管理->选项->系统对话框配置，如果你偏好在文件菜单查看最近文件或作为一个子菜单，也同样可以在这里配置。

保存

保存所有更改到当前保险箱。如不保存前面所作的更改或数据库被打开为只读，那么所作改动将被丢弃。

另存为

保存当前保险箱的一个副本在一个新名称或位置下。

输出

输出当前数据库内容为一个外部文件任一旧 Password Safe 数据库格式 XML 或文本文件。查看这里获得更多有关输出数据库为其它格式的信息。

输入

输入一个以前输出的内容或正确格式化的文本文件、XML 或一个公共应用程序保存的密码数据库到当前数据库里。 查看这里获得更多有关输入文本文件的信息。

合并

合并一个 Password Safe 数据库或备份文件(.bak)到当前数据库里。只读模式禁止合并。

比较

比较当前打开的数据库与其它数据库。

备注: 合并与比较操作两者都可能耗费较长时间，尤其是大数据库。

属性

显示当前被打开的 password safe 数据库有关信息。例如: 版本格式、组数量、项目数量以及最后保存时间并且由谁所作的保存等相关信息。

退出

关闭 Password Safe 程序。如果对数据库所作的改动一直未给予保存，关闭时将被提示保存数据库。
