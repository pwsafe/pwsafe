Password Safe

你有多少用户名和密码需要跟踪保存呢？无论答案是一个或数百个，Password Safe 都能让你轻松、安全地建立一个安全并加密的主密码清单，以及提供一个单独的主密码解除所有锁定的密码清单数据库（保险箱）。

安全始于用户！保留书面密码清单在便条纸或在桌面的文本文档上都是极为不安全的，并很容易被窥视（基于网络和人两者）。许多人使用相同密码一遍又一遍的在广泛的系统和网站用来用去，长此下去也许噩梦将不期而至。比如一旦有人算出一个密码，他们将能算出你的所有密码，并立刻有机会获得你生活中的每一部份（系统、电子邮件、零售、财务、工作）。

Password Safe 允许用户管理旧密码和轻松快捷的生成、存储、组织、检索及使用复杂的新密码；利用密码策略控制，一旦存储，每次仅需点击几下用户名和密码便可离开。

利用 Password Safe 用户能够使用自定义参考组织密码，例如：用户ID、类别、网站或位置。并可以选择存储所有密码在单独加密的主密码清单（高强度加密的密码数据库）中或使用多个数据库进一步组织密码(例如：工作和家庭)，相信它直观的界面和快速运行等便捷优点，能成为你装机必备的优秀软件。

Password Safe 的 windows 版本是一个完全免费与开源的多语言版软件。软件以 C++ 开发而成，其严谨简洁的语法，令本程序运行迅速、占用资源少，在众多密码管理软件群中，可谓出类拔萃。其利用 Twofish 算法为你加密的数据库，暂时无人能破解（包括程序开发者），所以请一定记得别忘了你的数据库主密码，不然没有任何人能帮助您找回已存的所有密码信息。本软件绝对不包含任何恶意插件、后门、更不包含木马病毒等代码，请您完全放心使用！
