显示选项卡

显示选项卡功能将控制 Password Safe 与密码项目如何显示。

总是保持 Password Safe 窗口在最前

如勾选该项，最大化时的 Password Safe 将总是保持窗口在监视器最前端，不管其它窗口的焦点。

列表与树状目录视图时显示密码

如勾选该项，在树状目录 视图展开时，项目关联的用户名密码将被显示在栏目中。用户名被标识在方括号中，密码则被花括号标识。收缩列表窗口则不可见。

添加与编辑时显示密码

如勾选该项，密码总是被隐藏在添加项目与编辑/查看项目对话，默认状态为可见。 在对话框单击“显示/隐藏”按钮，可以为指定的项目显示或隐藏密码。

编辑时显示备注内容

如勾选该项，备注内容将被默认显示在编辑对话；如不勾选，为了查看文本将需要单击备注区域。

在树状目录首位放置分组

默认状态，项目被绝对按字母顺序排序在树状目录视图。当该选项被选中时，所有组将被显示在首行(按字母顺序排序)，接着为所有最高级别的项目。

在列表显示网格线

如勾选该项，突出网格线将被显示在显示列表，使它看起来很像电子表格。

提前 N 天通知密码失效

如勾选该项，PasswordSafe 将在读取一个数据库时扫描所有项目 (在典型启动时) 并显示一列从当前日期开始至失效日期是 N 天或更少。

初始树状目录视图

该项确定当打开数据库时，项目将如何显示在树状目录视图。

初始系统托盘图标

该选项允许更改当 Password Safe 开始启动和在还未打开数据库时系统托盘图标显示的颜色。
