用户名、密码和组: 附加

用户名、密码和组连同附加参数被安排在四个选项卡:

  • 基本: 所有密码管理的必要元素。
  • 附加: 附加的规则，包括历史密码。
  • 日期和时间: 密码失效日期与统计信息。
  • 密码策略: 指定有别于默认的密码策略。

附加 选项卡


  • 自动输入: 如果该字段未空，它的内容将替换为自动输入函数的默认值。需要了解更多信息，请查看自动输入帮助主题。
  • 运行命令: 该字段可被用于指定一个命令或通过操作“运行命令”运行程序（通过编辑菜单，右击弹出菜单或快捷键 Ctrl-R）。 Password Safe 可以传递自变量到命令行，如：项目的用户名，密码等。 所述查看这里。
  • 双击项目: 该选项允许你定义当你双击一个 PasswordSafe 项目时将产生何种动作。 默认状态下，双击为复制项目密码到剪贴板。默认操作设定在管理->选项->杂项。
  • 历史密码: 该选项可以配置 Password Safe 保存项目的最后几个历史密码。这里可以在你创建一个项目或查看一个现有项目时完成其中的配置。 你可以查看项目最近相关的历史密码，也可以更改被保存的最近历史密码数量。欲了解更多信息，请查看历史密码页。
