Una nota sobre contraseñas maestras

Una de las claves más importantes para el mantenimiento de tu seguridad es seleccionar una contraseña maestra fuerte para tu base de datos Password Safe. La base de datos ("Segura") contiene todos tus usuarios, contraseñas y cualquier información adicional que hayas añadido. Escoger el nombre del cónyuge, un cumpleaños, o alguna otra combinación fáciles de adivinar hace sus datos vulnerables. Es importante que elijas una contraseña maestra difícil de adivinar que puedas recordar con facilidad.

Password Safe comprueba si la contraseña maestra es débil. Por lo general, una contraseña maestra (como cualquier contraseña) debe estar compuesta de una mezcla de mayúsculas y minúsculas con al menos un numero o carácter especial (como los que se encuentran por encima de las teclas numéricas en el teclado). Si se introduce una contraseña maestra que el programa considera débil, se le solicitará confirmación. Tienes la opción de aceptar la contraseña introducida (débil) o introducir una nueva.

Para cambiar tu contraseña maestra, consulta Cambiar las contraseñas maestras.

Nota: Password Safe no tiene un mecanismo para recuperar contraseñas de seguridad perdidas u olvidadas.
Esta es una decisión de diseño deliberada, ya que cualquier mecanismo para recuperar las contraseñas perdidas plantea un agujero de seguridad que pueden ser explotado.
