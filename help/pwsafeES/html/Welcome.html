Password Safe

¿Cuántas contraseñas y nombres de usuario tienes que recordar? Tanto si la respuesta es una o cientos, Password Safe te permite crear de una forma fácil y segura lista encriptada de contraseñas segura. Una única contraseña maestra desbloquea a todas para su uso.

La seguridad empieza con el usuario. Mantener listas de contraseñas escritas en pedazos de papel, o en un documento de texto en el escritorio es inseguro y está a la vista de los ojos curiosos (humanos y de software). Utilizar la misma contraseña una y otra vez en un amplio espectro de sistemas y sitios web crea un escenario de pesadilla en la que una vez que alguien ha descubierto una contraseña, se han descubierto todas sus contraseñas y ahora tienen acceso a cada parte de tu vida (sistema, correo electrónico, comercio minorista, finanzas, trabajo).

Password Safe te permite administrar tus viejas contraseñas y generar, almacenar, organizar, recupuerar y usar nuevas y complejas contraseñas de una manera fácil y rápida, utilizando políticas de contraseñas que controlas. Una vez almacenadas, tus usuarios y contraseñas están a unos clic de distancia.

Usando Password Safe puedes organizar tus contraseñas utilizando referencias personalizables —por ejemplo, por identificador de usuario, categoría, sitio web or ubicación. Puedes optar por almacenar todas tus contraseñas en una única lista encriptada de contraseña maestra (una base de datos de contraseñas encriptada), o utilizar múltiples bases de datos para organizar tus contraseñas (trabajo y casa, por ejemplo). Todo esto con una interfaz intuitiva que estará lista y funcionando en minutos.
