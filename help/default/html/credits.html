Program Credits

  • Grateful thanks to Bruce Schneier for creating the Twofish and Blowfish algorithms and for releasing Password Safe to the public; and to SourceForge.net for hosting the project and supporting the Open Source community.
  • Password Safe version 3.12 and later uses an algorithm created by Tom Van Vleck, Morrie Gasser, and Dan Edwards to generate pronounceable passwords.
  • Password Safe version 3.29 and later use pugixml for parsing the application's configuration file. The code for pugixml is provided under the MIT license.
  • The wxWidgets build of Password Safe from version 1.12 and later use graphics from PNG Repo provided under the Creative Commons 4.0 license.
