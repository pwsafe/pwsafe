File Menu

The File Menu contains options related to managing and creating new Password Safe databases.

New

Create a new blank password safe database.

Open

Open an existing password safe database.

Close

Closes the currently open password safe database without exiting the application.

Recent Safes

This section of the file menu may list the recently opened database files. This section will be blank if this is the first time Password Safe has been used.

Notes:

  • The list of recently opened files may be cleared by selecting "Clear Recent Safe List"
  • The number of recent files that Password Safe will keep may be configured via the Manage->Options->System dialog box, where you may also configure if you prefer to see the recent files in the File menu, or as a submenu.

Save

Save any changes to the current safe. If no changes have been made since the previous save, or if the database was opened read-only, this choice will be greyed-out.

Save As

Save a copy of the current safe under a new name or location.

Export To

Export the contents of the current database to an external file in either an old Password Safe database format, XML, or as text. See here for more information on exporting the database to other formats.

Import From

Import the contents of a previously exported or correctly formatted text file, XML, or a KeePass password database into the current database. See here for more information on importing text files.

Merge

Merge a Password Safe database or backup file (.bak) into the current database. Disabled in read-only mode.

Compare

Compare the currently opened database with another.

Note: Both the Merge and Compare operations may take a long time, especially for large databases.

Properties

Display information about the currently opened password safe database: Database format version, number of groups, number of entries, and information about when it was last saved and by whom.

Exit

Closes the Password Safe program. If changes were made to the database since the last save, you will be prompted to save the database.
