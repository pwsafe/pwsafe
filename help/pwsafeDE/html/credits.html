Programm Danksagung

Herzlichen Dank an Bruce Schneier für das Erschaffen des Twofish Algorithmus und für die Freigabe von Password Safe an die Allgemeinheit, und an SourceForge.net für das Hosting des Projektes und Unterstützung der Open Source Gemeinschaft.
Password Safe verwendet ab Version 3.06 TinyXML zum Parsen der Konfigurationsdatei dieser Anwendung. Der Kode für TinyXML wird unter der zlib license zur Verfügung gestellt.
Password Safe verwendet ab Version 3.12 einen Algorithmus erstellt von Tom Van Vleck, Morrie Gasser und Dan Edwards, um aussprechbare Passwörter zu generieren.

  • Entwicklungsteam: einige der Leute die bei der Entwicklung mitgemacht haben und dabei helfen Password Safe am Leben zu erhalten.
  • Password Safe Forum: Ein Forum für Fragen, Unterstützung und Probleme.
  • Password Safe News: RSS feeds verfügbar für dieses Projekt.
  • Seite für die Mailingliste von Password Safe, niedriges Aufkommen (ungefähr eine Meldung pro Monat oder so), Spam freie Seite für Ankündigungen und Mailingliste.
