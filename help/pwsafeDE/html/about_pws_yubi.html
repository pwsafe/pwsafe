Password Safe und YubiKey

YubiKey ist ein innovatives Authentifizierungsgerät in Hardware eingebettet, entwickelt und vertrieben von Yubico. Für den Anwender sieht es aus, wie ein kleiner, dünner USB-Adapter mit einer Taste. In Password Safe funktioniert es wie ein Authentifizierungs Token, der eine große hardwaremäßige Sicherheit für die Datenbank darstellt.

YubiKey beschaffen

Zu Beginn benötigen Sie natürlich ein YubiKey Gerät. Diese können Sie online hier erhalten. Beachten Sie bitte, dass Password Safe einen YubiKey mit einer Firmwareversion von mindestens 2.2 voraussetzt.

Konfiguration des YubiKey

Der YubiKey muss richtig konfiguriert werden, um den Betrieb mit Password Safe zu gewährleisten. Dies erfolgt im Dialogfenster unter Verwalten->Yubikey... , wie im Kapitel Verwalten beschrieben.
Wenn Sie schon einen YubiKey haben, achten Sie darauf, dass Password Safe Slot 2 für die Authentifizierung verwendet, und dabei Slot 1 für andere Anwendungen frei lässt.

Verwendung des YubiKey

Sobald der YubiKey konfiguriert ist, können Sie ihn zusammen mit dem Master-Passwort verwenden, um die Password Safe Datenbank zu schützen, wie in Neue Datenbank erstellen beschrieben.
