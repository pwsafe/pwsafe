Использованные программы

Выражаем благодарность Брюсу Шнайеру (Bruce Schneier) за создание алгоритма шифрования Twofish и публикацию Password Safe, а также SourceForge.net за размещение проекта и поддержку сообщества Open Source.
С версии 3.29 для работы с файлом настроек Password Safe использует pugixml. Код pugixml распространяется под лицензии MIT.
Начиная с версии 3.12, Password Safe может создавать произносимые пароли с помощью алгоритма Тома ван Флека (Tom Van Vleck), Морри Гассера (Morrie Gasser), и Дэна Эдвардса (Dan Edwards).
