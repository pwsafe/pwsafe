Вкладка «Разное»


На вкладке Разное настраиваются действия и значения по умолчанию.

Подтверждать удаление элементов

Если помечено, при удалении элемента будет выводится запрос подтверждения.

Сохранять время последнего доступа к записи

Если помечено, Password Safe будет записывать время доступа к элементу (копирование пароля, просмотр данных элемента и пр.). Время хранится вместе с элементом и может быть просмотрено в диалоге редактирования/просмотра элемента.

Примечания:
  • В данном случае контейнер будет изменяться даже если элементы не редактировались. Поэтому, при закрытии контейнера вам будет предложено сохранить изменения, даже если вы ничего явно не изменяли.
  • Если контейнер открыт в режиме «только чтение», время последнего доступа не обновляется.

Закрывать приложение по Esc

Если помечено, клавиша Esc закрывает Password Safe. Обратите внимание, что если активна настройка «Отображать значок в области уведомлений», то клавиша Esc лишь скрывает окно. Его можно открыто двойным щелчком по значку в области уведомлений.

По двойному клику

Позволяет установить действие, которое Password Safe будет выполнять при двойном щелчке на элементе. Действием по умолчанию является копирование пароля в буфер обмена. Возможные следующие действия:

  • Автонабор
  • Перейти по ссылке
  • Перейти по ссылке и выполнить автонабор
  • Копировать заметки в буфер обмена
  • Копировать пароль в буфер обмена
  • Копировать пароль в буфер и свернуть
  • Копировать имя пользователя в буфер
  • Выполнить команду
  • Отправить письмо
  • Просмотр/редактирование элемента

Обратите внимание, что поведение «Копировать пароль в буфер и свернуть» будет не таким, как ожидалось, если на вкладке Безопасность включена настройка «Очищать буфер обмена при сворачивании».

Автонабор

  • Свернуть после автонабора По умолчанию, Password Safe сворачивает окно при выполнении автонабора. Если флажок снят, вместо сворачивания Password Safe будет перемещать своё окно на задний план.
  • Строка автонабора по умолчанию Здесь можно указать строку автонабора, применяемую по умолчанию. При необходимости вы можете переопределить её в настройках элемента. Например, данную настройку можно использовать для изменения задержки автонабора всех элементов. Для получения дополнительной информации об автонаборе см. здесь.
  • Задержка между символами Определяет время ожидания (в миллисекундах) перед автонабором следующего символа в последовательности. Значение по умолчанию (10 мс) подходит для большинства систем. Но вы можете уменьшить его (сделать автонабор быстрее), если у вас мощный компьютер, или, наоборот, увеличить (замедлить автонабор) для старых машин. Описание того, как настроить задержку ввода для конкретного элемента, приведено в разделе Автонабор.

Имя пользователя по умолчанию

Данная настройка позволяет указать имя пользователя, подставляемое при создании нового элемента. Если помечено, указанное имя будет по умолчанию подставляться в новые элементы. При создании элемента вы можете изменить это имя.

Альтернативный браузер

Указанный здесь браузер будет использован при выполнении команды «Перейти по ссылке», если в настройках элемента ссылка начинается с «[alt]», например, «[alt]http://www.foo.net». Данный функционал может потребоваться, например, если в качестве браузера по умолчанию настроен Firefox, а связанный с элементом сайт лучше работает в Internet Explorer. В этом случае необходимо указать Internet Explorer как альтернативный браузер («C:\Program Files\Internet Explorer\iexplorer.exe»).

Параметры командной строки для браузера указываются в поле «параметры командной строки». Не заключайте значение в кавычки, если это не требуется для выбранного браузера.

Внешний редактор

Указанный здесь внешний редактор будет доступен через контекстное меню при вставке/редактировании элемента. Для получения дополнительной информации см. Имена пользователей, пароли и группы.

Параметры командной строки для редактора указываются в поле «параметры командной строки». Не заключайте значение в кавычки, если это не требуется для выбранного редактора.
