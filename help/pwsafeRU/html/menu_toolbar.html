Панель инструментов

Доступно два стиля отображения значков на панели инструментов: «современный» и «классический». Для каждого значка на панели инструментов доступны всплывающие подсказки. Для отображения подсказки достаточно навести мышку на значок. Для удобства ниже приведено краткое описание значков.

Начиная с версии 3.11 можно изменить набор значков и порядок их отображения на панели.

Переключение между стилями отображения значков

  1. Откройте меню Вид
  2. Выберите Классическая панель инструментов или Современная панель инструментов.
