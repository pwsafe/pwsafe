Мастер выбора файла или контейнера

Сравнение, слияние или синхронизация

При сравнении (слиянии, синхронизации) контейнеров требуется указать имя контейнера, который нужно сравнить (объединить, синхронизировать) с открытым в настоящий момент контейнером.

Запрашиваемый пароль относится к этому контейнеру, а не к уже открытому.

Экспорт в простой текст или XML

При экспорте текущего контейнера в простой текст или XML требуется указать имя файла, в который будут записаны данные (если указанный файл существует, будет выдан запрос подтверждения перезаписи).

Запрашиваемый пароль относится к открытому контейнеру.

Чтобы в последующем файл можно было корректно импортировать, нужно указать символ, используемый для разрыва строк в заметках. При экспорте в простой текст этот символ также используется для замены точек (.) в заголовке. Это сделано для того, чтобы при импорте можно было восстановить разбиение на группы, т. к. точка используется для отделения групп.

Расширенные условия отбора

Во всех случаях можно поставить флажок «Расширенные условия отбора». После этого будет доступна ещё одна страница, на которой можно указать ограничения по группе, заголовку, имени пользователя, ссылке или заметкам, а также отобрать поля для сравнения, слияния или экспорта.

Если вы передумаете, для экспорта всех элементов можно нажать кнопку «Назад» и снять флажок «Расширенные условия отбора».

Примечания:
  • Ограничения по группе, заголовку, имени пользователя, ссылке или заметкам недоступны, если через контекстное меню или меню «Правка» экспортируется один элемент.
  • При слиянии контейнера с текущим невозможно указать, какие поля элемента будут скопированы. Если элемент попадает под условие отбора, он целиком помещается в текущий контейнер.
