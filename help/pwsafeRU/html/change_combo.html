Для изменения мастер-пароля откройте контейнер, пароль которого требуется изменить. После этого в меню Сервис выберите пункт Изменить пароль для контейнера. Здесь необходимо ввести текущий мастер-пароль, а затем, новый пароль (вводится дважды, для подтверждения). Для ввода пароля можно использовать физическую или экранную клавиатуру.

Password Safe имеет встроенные средства проверки сложности мастер-пароля. Мастер-пароль (как и любой другой) должен состоять из букв разного регистра, содержать хотя бы одну цифру или специальный символ (например, один из тех, что расположены над цифрами на клавиатуре). Если программа решит, что выбранный пароль является слабым, будет выдан запрос подтверждения. Вы можете подтвердить использование введённого «слабого» пароля, или ввести новый пароль.

Для тех, кто использует YubiKey: в дополнение к функции смены мастер-пароля, этот диалог позволяет добавить или убрать аутентификацию при помощи YubiKey, установить или убрать пароль, используемый вместе с YubiKey, или привязать к контейнеру другой ключ YubiKey. Ниже приведены инструкции по выполнению перечисленных действий.

Добавление аутентификации при помощи YubiKey

Чтобы изменения способа аутентификации при доступе к контейнеру Password Safe с мастер-пароля на YubiKey:
  1. Введите текущий мастер-пароль в поле «Старый пароль».
  2. Подключите YubiKey
  3. Если требуется, введите (и подтвердите) новый пароль в поля «Новый пароль» и «Подтверждение».
  4. Нажмите на кнопку с надписью YubiКey, расположенную рядом с полем Новый пароль и коснитесь кнопки на устройстве YubiKey.

Удаление аутентификации при помощи YubiKey

Для того, чтобы изменить параметры аутентификации при доступе к контейнеру на использование только мастер-пароля:
  1. Подключите YubiKey.
  2. Если для доступа к контейнеру также используется пароль, введите его в поле «Старый пароль».
  3. Нажмите на кнопку с надписью YubiКey, расположенную рядом с полем Старый пароль и коснитесь кнопки на устройстве YubiKey.
  4. Введите новый пароль и его подтверждение в соответствующие поля.
  5. Нажмите OK.

Добавление/удаление/изменение мастер-пароля

Password Safe может использовать YubiKey как единственны способ аутентификации, так и как дополнительны к обычному паролю. Для переключения между этими режимами:
  1. Подключите YubiKey.
  2. Если для доступа к контейнеру также используется пароль, введите его в поле «Старый пароль».
  3. Нажмите на кнопку с надписью YubiКey, расположенную рядом с полем Старый пароль и коснитесь кнопки на устройстве YubiKey.
  4. В полях «Новый пароль» и «Подтверждение» установите пустое значение, чтобы отключить проверку пароля, или новый пароля чтобы изменить или добавить проверку пароля.
  5. Нажмите на кнопку с надписью YubiКey, расположенную рядом с полем Новый пароль и коснитесь кнопки на устройстве YubiKey.

Изменение YubiKey

Для привязку другого YubiKey:

  1. Подключите старый YubiKey.
  2. Если для доступа к контейнеру также используется пароль, введите его в поле «Старый пароль».
  3. Нажмите на кнопку с надписью YubiКey, расположенную рядом с полем Старый пароль и коснитесь кнопки на устройстве YubiKey.
  4. Отключите старый YubiKey и подключите новый.
  5. Если требуется, введите (и подтвердите) новый пароль в поля «Новый пароль» и «Подтверждение».
  6. Нажмите на кнопку с надписью YubiКey, расположенную рядом с полем Новый пароль и коснитесь кнопки на устройстве YubiKey.

Примечания:
  • В Password Safe не существует механизма для восстановления потерянных (забытых) мастер-паролей.
    Это осознанное решение разработчиков, так как любой механизм восстановления потерянных паролей является брешью в системе защиты и может быть использован злоумышленниками.
  • Всегда выбирайте пароли, которые сложно угадать. Простой пароль, такой как «пароль» или ваше имя, легко подбирается и сводит на нет защиту, обеспечиваемую Password Safe.
