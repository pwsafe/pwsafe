Политика создания паролей

Политика паролей описывает правила, используемые при создании паролей: буквы, их регистр, длина паролей и использование спецсимволов. В некоторых случаях требуется создавать пароли, содержащие только строчные буквы, в других требуется наличие прописных букв, цифр и спецсимволов, а также указание минимальной длины. В общем случае лучше всего использовать пароли максимально допустимой длины, состоящие из смеси символов различных наборов. На всякий случай проверяйте соответствие созданного пароля указанным требованиям.

Свой набор символов для создания пароля можно задать через меню Сервис→Политики создания паролей.... Кроме того, каждый элемент может иметь свой набор символов для пароля, задаваемый в диалоге добавления/редактирования элемента. Примечание. Эти символы не используются при создании произносимых или удобочитаемых паролей.

Диалог Управление политиками создания паролей позволяет задавать глобальные правила создания паролей. Все настройки политик сохраняются в заголовке контейнера и поэтому привязаны к нему. Данная политика и пароль могут быть переопределены в диалоге редактирования/просмотра элемента.

Выбор правил создания паролей

Каждый флажок (например Цифры, Строчные буквы и т. д.) заставляет Password Safe включать указанные символы в создаваемый пароль. Созданный пароль будет иметь указанную длину (ни больше, ни меньше). Единственное исключение — настройка Только шестнадцатеричные цифры (0–9, a–f). Если флажок установлен, будут использованы только символы, представляющие шестнадцатеричные цифры. Установка данного флажка блокирует другие настройки политики.

Выбор Создавать произносимые пароли приведёт к тому, что Password Safe будет создавать бессмысленные, но произносимые на английском языке слова, например, «ingstria», «ringsony», и пр. Данная опция может быть удобна в случае, если требуется создать стойкий пароль, который легко запомнить без Password Safe. Указание данной настройки вместе с Цифры и/или Символы приведёт к замене некоторых букв сходными по написанию знаками («leet-эквивалентами»), например, «E» может быть заменено на «3», «S» на «5», и т. д.

Если поставлен флажок Символы, к алфавиту добавляются следующие символы: + - = _ @ # $ % ^ & ; : , . < > / ~ \ [ ] ( ) { } ? ! |. Если установлен флажок Только удобочитаемые символы, набор символов сокращается до + - = _ @ # $ % ^ & < > / ~ \ ?

При установленном флажке Только удобочитаемые символы набор букв сокращается до: a b c d e f g h i j k m n o p q r s t u v w x y z и A B C D E F G H J K L M N P Q R T U V W X Y, а набор цифр ограничивается 3 4 6 7 8 9.
