Комбинации клавиш/горячие клавиши

Ниже описываются особенности определения и использования горячих клавиш в Password Safe.

Указанные комбинации клавиш проверялись на 102-клавишной клавиатуре с расширенной английской раскладкой. Стандартные комбинации клавиш должны работать на любой другой клавиатуре с латинскими символами. Некоторые комбинации клавиш могут отличаться, например, на клавиатуре с английской раскладкой символ «#» находится на клавише с тильдой «~», а на американской раскладке он находится на той же клавише, что и символ 3 (для вызова используется Shift+3). Просмотрите таблицу и, при необходимости, адаптируйте её для вашей клавиатуры (в любой момент вы можете проверить, работает ли та или иная комбинация).

В зависимости от языка, некоторые комбинации с клавишей Alt могут быть зарезервированы. Это буквы, используемые для вызова элементов главного меню. При нажатии клавиши Alt в главном окне, соответствующие буквы в именах пунктов меню будут подчёркнуты. Для Английского языка это Alt+F (File), Alt+E (Edit), Alt+V (View), Alt+M (Manage) и Alt+H (Help). Для прочих локализаций нужно свериться с тем, что будет подчёркнуто в главном меню. Например, в русской локализации пункт меню «File» заменяется на «Файл», поэтому вместо Alt+F будет зарезервирована комбинация Alt+Ф.

Клавиша Enter также зарезервирована, так как она используется для задания горячих клавиш на вкладке Сервис→Настройки→Комбинации клавиш. Поэтому её невозможно переназначить. Также, при отображении элементов в виде дерева, действие «Развернуть/свернуть группу» вызывается по клавише Enter. Несмотря на то, что на данное действие можно было бы назначить другую комбинацию клавиш, сворачивание/разворачивание ветви всё равно происходило бы и по нажатию клавиши Enter, так как это системная комбинация для древовидного списка. По этой причине комбинацию клавиш для данного действия нельзя задать на вкладке «Комбинации клавиш».

Клавиша Esc также зарезервирована, так как она используется для скрытия панели поиска и сворачивания Password Safe в значок (или выхода из приложения, если использование значка в области уведомлений отключено в настройках). Кроме того, нажатие на Esc равносильно нажатию кнопки «Отмена» в диалогах.

По умолчанию, горячая клавиша для просмотра в режиме «только чтение» совпадает с клавишей редактирования в обычном режиме. Данные сочетания нельзя задать по отдельности. Аналогично, «Удалить элемент» совпадает с «Удалить группу», а «Переименовать элемент» совпадает с «Переименовать группу». Поэтому, соответствующие действия на вкладке «Комбинации клавиш» названы просто «Удалить» и «Переименовать».

Некоторые комбинации клавиш зарезервированы ОС Windows, например, Ctrl+Alt+Delete, Tab, Shift+Tab, Alt+Tab, Alt+Shift+Tab и др.

Нестандартные клавиши не поддерживаются (они присутствуют на клавиатуре с раскладкой Kanji).

Password Safe не различает правые и левые клавиши Ctrl, Alt и Shift. Поэтому, на некоторых клавиатурах часть комбинаций клавиш может быть недоступна. Например, символ евро на 102-клавишной английской расширенной клавиатуре набирается с помощью правого Alt (Alt Gr) и кнопки с цифрой 4. Но нажатие данной комбинации будет воспринято как Ctrl+Alt+4. Это связано с особенностями обработки комбинаций горячих клавиш в ОС Windows и не является ограничением, внесённым Password Safe.

Примечание. Использование нестандартных клавиш может привести к тому, что соответствующие горячие клавиши будут недоступны на клавиатурах, где данные символы отсутствуют. Поэтому, рекомендуется использовать только перечисленные ниже сочетания.

Примечания:

  1. Так как клавиша Enter используется для задания горячих клавиш, её нельзя назначить на то или иное действие.
  2. Ctrl+Q и Alt+F4 зарезервированы для закрытия приложения.
  3. F1 зарезервировано для вызова справки.
  4. Может зависеть от клавиатуры.
