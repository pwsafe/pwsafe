Имена пользователей, пароли и группы: дополнительно

Имена пользователей, пароли и группы, а также другие параметры настраиваются на четырёх вкладках:

  • Основные: все необходимые элементы, связанные с управлением паролями
  • Дополнительно: дополнительные характеристики, включая историю паролей
  • Дата и время: дата истечения пароля и статистика
  • Политика создания паролей: переопределяет для данного элемента «политику по умолчанию»

Вкладка «Дополнительно»


  • Автонабор: если поле не пусто, его содержимое заменяет на параметры автонабора, установленные по умолчанию (значение по умолчанию указано сверху и берётся из настроек Password Safe для текущего контейнера, задаваемых на вкладке Сервис→Настройки→Разное). Для получения дополнительной информации см. Автонабор.

  • Выполнить: данное поле используется для указания команды/программы, выполняемой при выборе пункта «Выполнить команду» (в меню «Правка» или контекстном меню) или по горячей клавише Ctrl+R. Password Safe может передать в качестве аргументов имя пользователя, пароль и содержимое других полей, описанных здесь.

  • Действие по двойному клику и действие по Shift+двойной клик: задаёт действие, которое Password Safe выполнит при двойном щелчке на элементе или при двойном щелчке на элементе с нажато клавишей Shift. Действиями по умолчанию являются копирование пароля в буфер обмена для двойного клика и копирование имени пользователя в буфер обмена для Shift+двойной клик. В выпадающих списках значение по умолчанию помечено фразой «(по умолчанию)». Изменить значение по умолчанию можно на вкладке Сервис→Настройки→Разное.

  • История паролей: Password Safe может сохранять несколько предыдущих значений пароля. Настройку можно задать при создании или редактировании элемента. В списке отображаются предыдущие пароли для указанного элемента. Количество сохраняемых предыдущих паролей может быть изменено позже. Для получения дополнительной информации см. История паролей.

  • Комбинация клавиш: элементам контейнера можно назначить комбинации клавиш («горячие клавиши»), которые позволят выбирать их сразу после нажатия такое комбинации, независимо от текущего положения в контейнере.

    Особенности работы с горячими клавишами:
    • Комбинации клавиш для элементов хранятся как часть данных элемента внутри контейнера. Но, помимо этого, Password Safe позволяет задавать горячие клавиши для пунктов меню. Область действия горячих клавиш для пунктов меню распространяется на все контейнеры, открываемые на данном ПК. Значения по умолчанию зависят от выбранного языка. Ручная настройка горячих клавиш для пунктов меню производится на вкладке Сервис→Настройки→Комбинации клавиш.
    • По этой причине, если вы выберете для элемента ту же комбинацию клавиш, что уже назначена для пункта меню, будет выведено предупреждение. Это не приведёт к конфликта настроек, если позже вы измените язык локализации приложения или установите другую комбинацию для пункта меню. Аналогично, конфликта может не быть при назначении горячих клавиш, но он может произойти позже, если вы измените язык или комбинации клавиш для пунктов меню.
    • Т. к. использование комбинации клавиш связанных с элементами контейнера не изменяет сам контейнер (в отличие от комбинаций для пунктов меню, которые могу быть назначены на удаление элемента), горячим клавишам для элементов был дан более высокий приоритет в случае конфликта.
    • Комбинация клавиш для элемента должна содержать хотя бы один из модификаторов Alt и Ctrl (с или без Shift). Если ни Alt, ни Ctrl не присутствуют в комбинации, для получения уникальной комбинации проверяются все комбинации модификаторов Ctrl, Alt и Shift (в порядке Alt; Alt+Shift; Ctrl; Ctrl+Shift, Alt+Ctrl; Alt+Ctrl+Shift). Например, комбинация «Shift + 6» может быть заменена на «Alt + Shift + 6». Но, «Ctrl + 6» не будет изменяться автоматически.
    • Комбинация клавиш отвязывается от элемента, если она уже назначена другому элементу и
      • Элемент был импортирован (из простого текстового файла или XML-файла). Примечание. Об этом также будет сообщено в журнале импорта.
      • Элементы был добавлен при слиянии контейнеров. Примечание. Об этом также будет сообщено в журнале слияния.
      • Элемент был «перетянут» из другого контейнера. При перетаскивании внутри текущего контейнера этого не произойдёт.
      • Элементы был скопирован из диалога результатов сравнения контейнеров.
      • Элементы был создан посредством дублирования группы или элемента.

Список используемых в Password Safe комбинаций клавиш для элементов отображается под списком горячих клавиш для пунктов меню на вкладке Сервис→Настройки→Комбинации клавиш. Для просмотра/редактирования связанного элемента щёлкните дважды на назначенной комбинации. Также вы можете включить отображение столбца с комбинации клавиш в режиме Простого списка и/или создать фильтр, отображающий только элементы с назначенными горячими клавишами.
