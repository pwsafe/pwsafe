Расширенные условия отбора

В мастере при сравнении, слиянии, синхронизации и экспорте в простой текст или XML

Чтобы задать ограничения сравнения, слияния, синхронизации или экспорта, вы можете установить флажок «Расширенные условия отбора» на первой странице мастера. Таким образом вы также сможете ограничить список полей для сравнения, синхронизации или экспорта.

Для каждого вида действий указанные условия запоминаются на время сессии и не зависят друг от друга. Если вы закроете Password Safe и перезапустите его, ограничения будут сняты, а списки полей будут иметь значения по умолчанию, характерные для данных функций.

В случае, если вы уже задали какие-либо условия, или ограничили список полей, но не хотите использовать их в данный момент, нажмите кнопку «Назад» и снимите флажок «Расширенные условия отбора». Указанные «Расширенные условия отбора» сохранятся, но не будут использованы.

Для возврата ограничений на условия по умолчанию нажмите кнопку «Сбросить». Сброс условий в одном из мастеров (например, для сравнения) не приведёт к сбросу указанных значений для остальных (слияния, синхронизации, экспорта).

С помощью кнопки на панели поиска

Нажав на панели поиска кнопку «Расширенные», вы сможете задать условия, ограничивающие поиск. Вы также можете ограничить список полей для рассмотрения.

Указанные условия поиска запоминаются на время сессии и не зависят от описанных выше. Если вы закроете Password Safe и перезапустите его, ограничения будут сняты, а списки полей будут иметь значения по умолчанию, характерные для поиска.

В случае, если вы уже задали какие-либо условия, или ограничили список полей, но не хотите использовать их в данный момент, нажмите кнопку «Отмена» или «ОК», а затем кнопку «Расширенные» на панели поиска (чтобы она перестала быть утопленной). Указанные «Расширенные условия отбора» сохранятся, но не будут использованы.

Для возврата ограничений поиска на условия по умолчанию нажмите кнопку «Сбросить». Сброс условий поиска не приведёт к сбросу указанных ограничений для остальных мастеров.

Синхронизация элементов при просмотре результатов сравнения контейнеров

Если при сравнении окажется, что данные в контейнерах не совпадают, будет выведено окно с результатами. Нажатие правой кнопкой мыши на букве «Y» приводит к отображению контекстного меню. Для элементов из текущего контейнера единственное доступное действие это редактирование данного элемента (или просмотр, если контейнер открыт в режиме «только чтение»). Для элементов из сравниваемого контейнера доступны просмотр, копирование в исходный контейнер и синхронизация.

Если выбрано действие «Синхронизировать», отображается диалог, позволяющий указать поля для синхронизации.

Для каждого вида действий указанные условия запоминаются на время сессии и не зависят друг от друга. Если вы закроете Password Safe и перезапустите его, списки полей будут содержать значения по умолчанию, характерные для данных функций.
