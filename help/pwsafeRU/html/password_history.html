История паролей

Password Safe может сохранять некоторое количество предыдущих паролей (и дату изменения) для данного элемента. Это позволяет показать соответствие политикам создания паролей (что может быть необходимым в некоторых случаях), а также вернуться к предыдущему паролю, если по какой-либо причине смена пароля не удалась. История изменения паролей доступна на вкладке «Дополнительно» диалога редактирования элемента.

На историю паролей влияют следующие параметры:

Хранить N предыдущих паролей

Если флажок установлен, для данного элемента будут сохраняться предыдущие пароли. Также указывается количество сохраняемых паролей.
Примечание. Снятие флажка не приводит к очистке истории. После этого Password Safe лишь перестаёт сохранять пароли при последующих изменениях.

Копирование пароля

Щелчок на строке списка приводит к копированию соответствующего пароля в буфер обмена.

Очистить

При нажатии на данную кнопку история паролей для этого элемента будет очищена.

Примечание. Фактически история паролей очищается только после нажатия на кнопку «OK». Если вы нажали кнопку «Очистить» по ошибке, нажмите «Отмена» для отклонения изменений.

Копировать все

При нажатии на данную кнопку вся история паролей будет скопирована в буфер обмена.
