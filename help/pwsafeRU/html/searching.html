Поиск указанного текста

Password Safe позволяет производить поиск элементов по заданному тексту. Это удобно, если вы не помните имя элемента, но знаете, что в нём содержится, например, элемент, содержащий в заметках имя «Маша».

Для поиска в открытом контейнере нажмите Ctrl+F (или «Правка→Поиск»). Над строкой состояния появится панель поиска, похожая на ту, что изображена на рисунке.

Для начала поиска достаточно ввести нужный текст и нажать Enter или щёлкнуть по значку с лупой, расположенному справа от поля ввода. Количество найденных элементов (если такие есть) будет отображено в правой части панели, а первый подходящий элемент будет выделен и получит фокус. Для перехода к следующему найденному элементу нажмите Enter (или кнопку с лупой). После того, как все элементы будут пройдены, Password Safe перейдёт на первый найденный элемент.

Нажатие на клавишу F3 повторяет поиск. Для перемещения по найденным элементам просто нажимайте F3. При одновременном нажатии Shift и F3, выделение перейдёт на предыдущий найденный элемент контейнера. То же самое произойдёт, если нажать на кнопку с лупой удерживая клавишу Shift.

  • Для закрытия панели поиска нужно нажать на «x» слева от поля ввода.
  • По умолчанию поиск является регистронезависимым. Для переключения между регистрозависимым и регистронезависимым поиском щёлкните по кнопке с надписью «AA». При переключении на регистрозависимый поиск надпись изменяется на «Aa». (При использовании «современных» значков на кнопке изображены буквы «TT».)
  • Если требуется указать дополнительные параметры поиска (ограничить подмножество элементов или выбрать отдельные поля элементов), щёлкните по значку «...».
  • Для создания отчёта о найденных элементах щёлкните по значку .
  • Для очистки поля с искомым текстом нажмите кнопку, на которой изображён ластик.
