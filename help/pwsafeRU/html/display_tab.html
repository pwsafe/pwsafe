Вкладка «Вид»

Параметры на вкладке Вид влияют на вид Password Safe и элементов контейнера.

Располагать Password Safe поверх всех окон

Если помечено, развёрнутый Password Safe будет находиться поверх всех окон, независимо от того, какое окно имеет фокус ввода.

Показывать имя пользователя в дереве

Если помечено, имя пользователя будет отображаться в квадратных скобках после имени элемента (в режиме древовидного списка).

Показывать пароль в дереве

Если помечено, пароль будет отображаться в фигурных скобках после имени пользователя (в режиме древовидного списка). Строго не рекомендуется использовать данный параметр, если во время работы возможен просмотр экрана посторонними людьми.

Показывать заметки во всплывающей подсказке

Если помечено, заметки будут отображаться при наведении курсора мыши на элемент.

Показывать пароль при добавлении и редактировании

Если помечено, скрытый по умолчанию пароль в диалогах добавления и просмотра/редактирования будет видимым. Вы можете отобразить или скрыть пароль текущего элемента, нажав в диалоге кнопку «Показать/Скрыть».

Показывать заметки при редактировании

Если помечено, в диалоге редактирования заметки будут видимыми по умолчанию. В противном случае, для отображения нужно будет щёлкнуть по полю ввода заметок.

При редактировании переносить заметки по словам

Если помечено, в поле заметок будет использоваться перенос по словам. Иначе, при недостатке ширины, будет отображаться горизонтальная полоса прокрутки.

Показать сетку списка

Если помечено, будут видны светлые линии сетки, и список станет похож на таблицу.

Размещать группы вверху дерева

По умолчанию в древовидном списке элементы отсортированы по алфавиту. Если данная опция активна, группы будут расположены вверху (в алфавитном порядке), а за ними пойдут остальные элементы соответствующего уровня.

Выделять изменённые элементы

Изменённый элемент до момента сохранения будет выделен цветом. Если на вкладке «Резервные копии» поставлен флажок «Сохранять контейнер после вставки или редактирования», элементы никогда не будут помечены, так как сохранение происходит немедленно.

Предупреждать за N дней до истечения пароля

Если помечено, при чтении контейнера (обычно при запуске) Password Safe будет просматривать все элементы и уведомлять, если пароль истекает не позднее чем через N дней.

Вид дерева при старте

Определяет, как отображать элементы при открытии контейнера в режиме древовидного списка.

Прозрачность окна

Выключено по умолчанию. После включения, для того, чтобы связанные настройки вступили в силу, и можно было настраивать уровень прозрачности, Password Safe должен быть перезапущен.

Если включено, указывает уровень прозрачности окна (в процентах). 0 % — нормальный вид (полностью непрозрачно), 100 % — невидимо (полностью прозрачно). Значения прозрачности более 50 % не поддерживаются. Примечание. Применяется только к главному окну. Все диалоги, окна с сообщениями и меню остаются непрозрачными.

Обычно значения от 20 % до 40 % являются наиболее предпочтительными.
