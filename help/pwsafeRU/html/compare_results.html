Результаты сравнения

Если при сравнении контейнеров обнаруживаются отличия, выводится окно с результатами. Примечание. Так как сравниваются лишь заданные пользователем поля, два отличающихся контейнера могут быть признаны одинаковыми, если они отличаются лишь в полях, исключённых из сравнения. По умолчанию все поля участвуют в сравнении. Если вы укажете свой список полей для сравнения, он будет использоваться только в текущем экземпляре Password Safe. То же правило применимо для всех мастеров, но соответствующие списки независимы:

  • Сравнение
  • Экспорт в простой текст (весь контейнер или один элемент)
  • Экспорт в XML (весь контейнер или один элемент)
  • Слияние
  • Синхронизация

Для отличающихся элементов списка возможны следующие действия:

  • Щелчок правой кнопкой мыши на элементе из текущего контейнера (первый столбец):
    • Редактирование элемента, если контейнер открыт в режиме «чтение/запись», и элемент не является защищённым.
    • Просмотр элемента, если контейнер открыт в режиме «только чтение», или элемент является защищённым.
  • Щелчок правой кнопкой мыши на элементе из сравниваемого контейнера (второй столбец):
    • Просмотр элемента (сравниваемый контейнер всегда открывается в режиме «только чтение», т. к. для него недоступна отмена последнего действия).
    • Копирование элемента в текущий контейнер, если он открыт в режиме «чтение/запись», и существующий элемент не является защищённым.
    • Синхронизация элемента в текущем контейнере с выбранным, если текущий контейнер открыт в режиме «чтение/запись», и существующий элемент не является защищённым.
    • Если различающийся элемент присутствует в обоих контейнерах, доступен пункт меню «Сравнить элементы...». При выборе данного пункта отображается список всех полей этих элементов с указанием отличающихся.

Кроме того, вы можете выделить несколько элементов из сравниваемого контейнера. Для пометки элемента щёлкните левой кнопкой мыши во втором столбце (для пометки второго и последующих элементов удерживайте Ctrl). По окончании выбора вы можете скопировать отмеченный элементы («Копировать все элементы в исходный») или синхронизировать их с соответствующими экземплярами из текущего контейнера («Синхронизировать все элементы»). Примечание. Синхронизация доступна только если все выбранные элементы присутствуют в текущем контейнере. Защищённый элемент не может быть выделен вместе с остальными, т. к. он не может быть изменён. Как и в основном списке, защищённый элемент помечается символом «#».
