Меню «Помощь»

Меню Помощь предоставляет доступ к различным ресурсам связанным с Password Safe.

Помощь (F1)

Открывает данный файл со справкой.

Посетить сайт Password Safe

Открывает домашнюю страницу Password Safe в браузере.

О Password Safe

Данный пункт меню отображает окно с информацией о версии и лицензии на распространение исходного кода. Также здесь вы можете проверить наличие новых версий Password Safe. Из соображений безопасности Password Safe закроет текущий контейнер перед проверкой информации о новой версии.
