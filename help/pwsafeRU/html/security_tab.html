Вкладка «Безопасность»

На данной вкладке можно изменить пользовательские настройки безопасности.

Очищать буфер обмена при сворачивании

Если помечено, при сворачивании окна программы будет произведена очистка буфера от данных, помещённых туда Password Safe (например, имя пользователя, пароль и пр.). В этом случае никто не сможет прочитать то, что было записано в буфер. Обратите внимание, что использование данной опции вместе со «Свернуть после копирования пароля» на вкладке Разное приведёт к неожиданным результатам (пароль будет скопирован в буфер и сразу же удалён оттуда).

Очищать буфер обмена при выходе

Если помечено, при закрытии окна программы будет произведена очистка буфера от данных, помещённых туда Password Safe (например, имя пользователя, пароль и пр.). В этом случае никто не сможет прочитать то, что было записано в буфер.

Подтверждать копирование элемента в буфер обмена

Если помечено, Password Safe каждый раз будет запрашивать разрешение на копирование пароля в буфер обмена.

Блокировка контейнера

Если контейнер блокируется по одной из указанных ниже причин, контейнер сохраняется, и все его данные удаляются из памяти. При очистке памяти удаляется и журнал отката/повтора действий, т. к. он может содержать защищаемую информацию. По этой причине после разблокировки контейнера недоступен откат действий, произведённых до блокировки.

  • Блокировать контейнер при сворачивании

    Если помечено, при сворачивании программы контейнер будет блокироваться. Для доступа к контейнеру после открытия окна нужно будет ввести мастер-пароль.

  • Блокировать контейнер при блокировке компьютера

    Если данная настройка включена, контейнер будет блокироваться при блокировке компьютера. Её можно использовать для предотвращения несанкционированного доступа к контейнеру лиц, имеющих доступ к рабочей станции.

  • Блокировать контейнер после

    Если включено, Password Safe будет блокировать контейнер по истечении указанного времени с момента выполнения последнего действия с программой.

«Перейти по ссылке» копирует пароль в буфер обмена

По умолчанию, при переходе по ссылке Password Safe копирует пароль элемента в буфер обмена, чтобы его сразу можно было вставить в нужное поле. Но некоторым сайтам требуются другие данные (или они не требуются вообще). После снятия флажка Password Safe перестанет помещать пароль в буфер обмена при переходе по ссылке.

Сложность перебора для мастер-пароля

Положение ползунка определяет количество вычислительных ресурсов, требуемых для открытия и сохранения контейнера Password Safe. Значение обычная предоставляет достаточный уровень стойкости к перебору без заметного замедления открытия контейнера на большинстве платформ. При увеличении этого значения учитывайте самый медленный компьютер, на котором будет использоваться данный контейнер.
