Password Safe и YubiKey

YubiKey это инновационное устройство аутентификации разработанное и продаваемое корпорацией Yubico. Для пользователя YubiKey на небольшую флешку с кнопкой на одной из сторон. Для Password Safe он выступает как маркер аутентификации (токен), добавляющий привязку к физическому устройству для защиты контейнера.

Приобретение YubiKey

Очевидно, что для работы в таком режиме вам необходимо иметь YubiKey. Его можно приобрести в интернет-магазине Yubico. Для работы Password Safe требуется устройство с версией прошивки 2.2 и выше.

Настройка YubiKey

Перед началом работы YubiKey должен быть настроен для взаимодействия с Password Safe. Требуемые действия выполняются в пункте меню «Сервис→YubiKey...» (описание доступно в разделе Сервис).
Для хранения своих данных Password Safe использует слот № 2, оставляя 1-й слот свободным для использования в других приложениях.

Использование YubiKey

После того, как вы настроили YubiKey, его можно использовать вместе с мастер-паролем для защиты контейнера Password Safe (описание приведено в разделе Создание нового контейнера).
