Edit Menu

Through the Edit Menu users can add, edit, use and organize user names and passwords.

Add Entry

Add a new user name and password.
See User Names, Passwords, and Groups for more information.

Edit/View Entry

Edit the selected user name and password.
See User Names, Passwords, and Groups for more information.

Delete Entry

Delete the selected user name and entry.

Rename Entry

Rename the selected entry.

Find Entry

Find an entry containing user specified search text.

Duplicate Entry

Duplicates the selected entry.

Add Group

Create a new password group (e.g., Banking Passwords).

Copy Password to Clipboard

Copy the current selection's password to the clipboard.

Copy User Name to Clipboard

Copy current selection's user name to the clipboard.

Clear Clipboard

Clear the clipboard's memory (erases the contents of anything that has been stored there, such as a password or user id)

Perform Auto Type

Automatically input user name and password into the selected input field.
See Using Auto Type for more information.
