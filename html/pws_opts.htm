Password Safe Options

The configurable options are divided into the following categories:
Display - options that affect how PasswordSafe is displayed
Security - options pertaining to the security of the application
Password Policy - Define the properties of automatically generated passwords
Username - Define a default username for new entries
Misc - Various settings

Display

Always keep PasswordSafe on top - When this is selected, PasswordSafe will always be visible.
Show Password in display list - When this is selected, a column is added in the list view, showing the password for each entry. This option is not recommended, as it opens a potential security hole.
Show password by default in edit mode - Selecting this option will cause passwords to be displayed in the Edit/View Entry dialog box. By default, a row of asterisks ('*****') is displayed unless the "Show Password" button is pressed.

Security

Clear the clipboard upon minimize or exit - If you check this option, the clipboard will be cleared when you minimize or exit PasswordSafe.

Lock password database on minimize - If you check this option, sensitive information such as your safe combination and the names of the items will be cleared from memory when Password Safe is minimized. When Password Safe is restored, you will be prompted for the Safe Combination.

Confirm password database save on minimize - If you check this option, Password Safe will remind you when it automatically saves the password database on minimize. Unchecked, the password database will be automatically saved.

Confirm item copy to clipboard - If you check this option, Password Safe will notify you when a password is copied to the clipboard. Once you've gotten used to how Password Safe works, you will probably want to uncheck this option.

Lock password database on workstation lock - If you check this option, PasswordSafe will be minimized, sensitive information such as your safe combination and the names of the items will be cleared from memory when your workstation is locked (for example, by hitting Ctrl-Alt-Del and choosing "Lock Computer"). When Password Safe is restored, you will be prompted for the Safe Combination.

Lock password database after N minutes idle time -  If you check this option, PasswordSafe will be minimized, sensitive information such as your safe combination and the names of the items will be cleared from memory if PasswordSafe is unused for N minutes (N can be any value between 1 and 120).When Password Safe is restored, you will be prompted for the Safe Combination.

Password Policy

Default password length - This field determines the length of the password generated by the "Generate Random Password" button in the Edit/View Entry and Add Entry dialog boxes. The minimum length is 4 characters, and the maximum, 1024. Note: If hexadecimal passwords are to be generated, then this number must be even.

Use lowercase letters - Check this option if you wish generated passwords to include lowercase letters.

Use UPPERCASE letters - Check this option if you wish generated passwords to include uppercase (capital) letters.

Use digits - Check this option if you wish generated passwords to include digits (0-9).

Use symbols (i.e., ,%,$, etc.) - Check this option if you wish generated passwords to include characters asides from letters and digits.

Use only easy-to-read characters (i.e., without "0" and "O") - Checking this option will result in generated passwords to be without characters that are visually similar, such as the digit zero and capital O, one and lowercase ell, etc. This is especially useful if you do not copy&paste passwords, but rather display them and enter them manually.

Use hexadecimal digits only (0-9, a-f) - Check this option if you wish generated passwords to only contain hexadecimal characters. The password length must then be an even number. This can only be selected if no other options are selected. Similarly, if selected, all other options are disabled.

Username

Use a default username - If this option is checked, the username entered in the field below will be automatically inserted into the username field in the Add dialog box. Also, if a username field is left blank, the default name will be inserted.

Query user to set username as default - If this option is checked, whenever you enter a username for a new item and there is no default username, you will be prompted to set it as the default username if desired. See above for more information about default usernames.

Misc

Confirm deletion of items - When this option is checked, PasswordSafe will confirm that you want to remove the selected item when you issue the Delete command.

Save database immediately after Edit or Add - Checking this option will cause PasswordSafe to write changes to the database immediately, rather than waiting for an explicit Save or Save As command.

Escape key exits application - When this option is checked, hitting the Esc key will cause PasswrdSafe to exit. When this option is cleared, the Esc key has no effect.
