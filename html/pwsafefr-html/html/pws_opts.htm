Options de Password Safe

Les options configurables sont organisées dans les catégories suivantes :
Display (Affichage) - options qui affectent comment PasswordSafe est affiché
Security (Sécurité) - options concernant la sécurité de l'application
Password Policy (Politique de Mot de passe) - Définition des propriété des mots de passe générés automatiquement
Username (Nom d'Utilisateur) - Définition de l'Utilisateur par défaut pour les nouveaux enregistrements
Misc (Divers) - Différents paramètres

Display (Affichage)

Always keep PasswordSafe on top
(Toujours conserver Password Safe au dessus) - Lorsque cette option est sélectionnée, PasswordSafe sera toujours visible.

Show Password in display list
(Afficher les mots de passe) - Lorsque cette option est sélectionnée, une colonne est ajoutée dans la vue liste, affichant le mot de passe de chaque enregistrement. Cette option n'est pas recommandée, car elle ouvre une brèche dans la sécurité.

Show password by default in edit mode
(Affichage par défaut du mot de passe en mode édition) - Sélectionner cette option pour afficher les mots de passe dans la fenêtre Edit/View Entry. Par défaut, une ligne d'astérisques (*****) est affichée sauf si le bouton Show Password est activé.

Security (Sécurité)

Clear the clipboard upon minimize or exit
(Effacer le Presse Papier lors de Quitter et Minimiser) - Si vous activez cette option, le Presse papier sera effacé lorsque vous quittez ou minimisez PasswordSafe.

Lock password database on minimize
(Fermer la base lors de Minimiser)- Si vous activez cette , des information sensibles telles que votre combinaison de coffre et le nom des enregistrements seront effacé de la mémoire lorsque Password Safe est minimisé. Lorsque Password Safe est rétabli, votre combinaison de coffre vous sera demandée.

Confirm password database save on minimize
(Confirmer la sauvegarde lors de Minimiser)- Si vous activez cette option, Password Safe vous rappellera automatiquement lorsqu'il sauve la base lorsque vous le minimiser. Désactivée, et la base sera automatiquement sauvegardée sans avertissement.

Confirm item copy to clipboard
(Confirmation de copie vers le Presse papier)- Si vous activez cette option, Password Safe vous préviendra lorsqu'un mot de passe est copié vers le Presse papier. Lorsque vous êtes habitués à Password Safe, vous désactiverez probablement cette option.

Password Policy (Politique de Mot de Passe)

Default password length
(Longueur par défaut)- Ce champ détermine la longueur des mots de passe générés automatiquement par le bouton Generate Random Password dans les fenêtres Edit/View Entry et Add Entry. La longueur minimum est de 4 caractères, et le maximum de 1024. Remarque : Si vous voulez générez des mots de passe hexadécimaux, alors ce chiffre doit être pair.

Use lowercase letters
(Utiliser des lettres minuscules)- Sélectionner cette option si vous souhaitez générer des mots de passe avec des lettres en minuscule.

Use UPPERCASE letters
(Utiliser des lettre majuscules)- Sélectionner cette option si vous souhaitez générer des mots de passe avec des lettres en majuscule.

Use digits
(Utiliser des chiffres)- Sélectionner cette option si vous souhaitez générer des mots de passe avec des nombres (0-9).

Use symbols (i.e., ,%,$, etc.)
Utiliser des symboles -exemple: %,$, etc.-)- Sélectionner cette option si vous souhaitez générer des mots de passe avec des caractères autres que des lettres et chiffres.

Use only easy-to-read characters (i.e., without "0" and "O")
(Utiliser uniquement des caractères faciles à lire -i.e. sans "0" et "O"-)- Sélectionner cette option permet de générer des mots de passe sans caractères qui sont visuellement semblables, tels que le chiffre zéro et la majuscule O, un et la minuscule l, etc. Ceci est spécialement utile si vous ne copiez pas les mots de passe, mais vous les affichez et ensuite les copiez manuellement.

Use hexadecimal digits only (0-9, a-f)
Utiliser uniquement des caractères hexadécimaux (0-9, a-f)- Sélectionner cette option si vous souhaitez générer des mots de passe qui contiennent uniquement des caractères hexadécimaux . La longueur des mots de passe doit être un chiffre pair. Cette option peut être activée si aucune autre option n'est sélectionnée. De même, lorsque elle est sélectionnée, cette option désactive les autres.

Username (Nom de l'Utilisateur)

Use a default username
(Utiliser un Nom d'Utilisateur par défaut)- Si cette option est activée, le nom de l'utilisateur entré dans le champ en dessous sera automatiquement inséré dans le champ username de la fenêtre Add dialog box. De plus, si le champ username est vide, le nom par défaut sera ajouté.

Query user to set username as default
(Demande de définir l'utilisateur par défaut)- Si cette option est activée, dès que vous entrez un utilisateur dans un nouvel enregistrement et qu'il n'y pas d'utilisateur par défaut, Password Safe vous demandera de définir un utilisateur par défaut si besoin. Voir les instructions si dessus pour plus d'information à propos de l'Utilisateur par défaut.

Misc (Divers)

Confirm deletion of items>
(Confirmer la suppression des enregistrements)- Lorsque vous activez cette option, PasswordSafe vous demandera de confirmer la chaque suppression d'un enregistrement sélectionné lorsque vous utilisez la commande Delete.

Save database immediately after Edit or Add
(Sauvegarder la base de données immédiatement après Edition ou Ajout)- En activant cette option, PasswordSafe écriera automatiquement tout changement dans la base de mots de passe, au lieu d'attendre explicitement les commandes Save ou Save As.
